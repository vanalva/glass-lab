Glass Lab + Pernia Glass

Minimal brutalism design system for a glass manufacturer. Two brands share one variable system: Glass Lab (dark) and Pernia Glass (light). Built on the LUMOS framework with Inter for headings and monospace for UI.

10 // Responsive

Responsive / Breakpointless Design

Philosophy

LUMOS prefers breakpointless design: fluid values (clamp), auto-fit grids, and wrapping flexbox that adapt continuously rather than snapping at breakpoints. Container query thresholds control grid switching.

Key Techniques

TechniqueClassesHow It Works
Auto-fit Gridu-grid-autofitColumns auto-size to fill available space
Auto-fill Gridu-grid-autofillColumns maintain size, empty tracks collapse
Wrapping Flexu-flex-horizontal-wrapItems wrap to next line when space runs out
Fluid Typographyclamp() in variablesFont sizes scale smoothly between 20rem-90rem viewport
Fluid Spacingclamp() in variablesSpacing tokens scale with viewport

Container Query Thresholds

ClassDescription

--_responsive---large: 1; /* Desktop: active */ --_responsive---medium: 0; /* Tablet: inactive at desktop */ --_responsive---small: 0; /* Mobile: inactive at desktop */ --_responsive---xsmall: 0; /* Small mobile: inactive at desktop */ /* These flip at each breakpoint via @media queries */
